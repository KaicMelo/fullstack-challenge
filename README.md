# fullstack-challenge

## BACKEND

Conexão MySQL:
- Criar um banco de dados
- Adionar as configurações do banco no ".env" na pasta raiz do backend

Executar os seguintes comandos na raiz ##backend:
- npm i
- npm run migrate
- npm run seed
- npm start

## FRONTEND

Executar os seguintes comandos na raiz ##web:
- npm i
- npm start
